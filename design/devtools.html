The Devtools bar

Loom Devtools rides along a running app as a bar. It names the project first, carries the branch the project sits on beside it, and keeps the tools to the right: an aim to point with, and the notes made so far. It collapses to a handle when it is in the way, and returns from it. Drawn here in the theme's own colours, alongside the real bar the overlay now serves on every catalog page.
